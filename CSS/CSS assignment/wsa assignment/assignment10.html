      Tags in HTML

      HTML tags are "keywords which are enclosed in an angular bracket."
      most of the tags are in pair(with opening and closing tag).

      table HTML5 tags
      Categories Description
      basic tags defines the basic page structure,line breaking,title,comments etc ex:<body>
      formatting tags defines the text formatting. ex:<b>
      media tags defines audio and video content. ex:<audio>
      list tags defines lists and menu items. ex:<ol>
